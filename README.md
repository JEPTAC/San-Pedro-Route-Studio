# San Pedro Route Studio Mobile PWA

Versión liviana para iOS y Android, lista para GitHub Pages.

## Por qué esta versión carga mejor en celular

- Los logos ya no están incrustados en base64 dentro del HTML.
- La vista inicial usa calles livianas.
- El satélite queda como opción manual.
- Zoom máximo seguro en 17 para evitar mosaicos vacíos.
- Panel inferior optimizado para pantallas pequeñas.
- Sin grabación WebM interna, porque iOS/Android suelen limitar `getDisplayMedia`.
- Compatible como PWA instalable desde el navegador.

## Publicar en GitHub Pages

1. Sube todos los archivos del ZIP a un repositorio.
2. Activa GitHub Pages desde Settings > Pages.
3. Abre la URL en el celular.

## Instalar en Android

Abre la URL en Chrome > menú de tres puntos > Agregar a pantalla principal / Instalar app.

## Instalar en iPhone

Abre la URL en Safari > botón Compartir > Agregar a pantalla de inicio.

## Nota

Para grabar video en celular, usa la grabadora nativa de pantalla de iOS o Android.
