# San Pedro Route Studio Mobile Turbo

Versión móvil optimizada para iOS y Android.

## Objetivo

Resolver lentitud, botones incómodos y mapa incompleto en celular.

## Cambios técnicos

- Vista inicial en Calles rápidas.
- Satélite realista liviano con una sola capa.
- Sin triple capa satelital en móvil.
- Zoom máximo 17 y reducción automática si fallan mosaicos.
- UI móvil real: barra inferior, panel deslizable, botones grandes.
- Menos redibujos del mapa.
- Animación de ruta con cámara desacoplada.
- Modo pantalla limpia para grabar con la grabadora nativa del celular.
- PWA instalable.
